Os usuários possuem: **ID, Shell e Raiz**

#### Entendo as permissões no arquivos e diretorios:

![[Screenshot from 2025-12-08 12-39-47.png]]

**d** significa que é um diretório
 **-** significa que é um arquivo
 **l** link simbólico

Após a primeira letra, as **três seguintes diz respeito** as permissões dos usuários sobre o arquivo ou pasta
As três seguintes são as permissões do **grupo**
E as três ultimas são as permissões para todos os outros

**r** - Ler
**w** - Escrever
**x**- Executar
