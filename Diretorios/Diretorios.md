**LOGS** - /var/log/

**/etc** - Este é o diretório padrão onde os arquivos de configuração do sistema são armazenados em sistemas Unix/Linux. Muitos pacotes de software colocam seus arquivos de configuração neste diretório para manter uma organização consistente e porque é o local tradicionalmente usado para armazenar configurações do sistema.

**/etc/passwd** - Este é o arquivo padrão que armazena informações sobre as contas de usuário no sistema. Cada linha do arquivo contém detalhes sobre uma conta de usuário, incluindo nome de usuário, identificador de usuário (UID), identificador de grupo (GID), diretório home, shell padrão e outras informações.

**$PATH** - Permite a execução de comandos sem a necessidade de conhecer a localização do executável.

**/var** - Armazena arquivos de logs

**/tmp** - Arquivos temporários usados por programas durante sua execução.
